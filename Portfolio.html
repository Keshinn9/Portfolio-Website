SmartAttend: Attendance System

An automated student attendance management system built to eliminate manual tracking errors.

The Blueprint:

A robust Java Swing and MySQL-backed attendance management system built to digitize academic record-keeping, replacing slow, manual paper logs with a high-speed local kiosk.

The Engine (Tech Stack):

Java • MySQL • Java Swing • MVC Architecture • DAO Pattern • FlatLaf UI

Key Engineering Highlights

Architecture Isolation: Implemented a strict Model-View-Controller (MVC) pattern and custom Data Access Object (DAO) layers to separate front-end GUI components from underlying database query scripts.

Automated Business Logic: Programmed strict conditional time-windows allowing logs only during precise schedules, with automated algorithms calculating attendance states such as Present, Late, or Absent.

UI Modernization: integrated the FlatLaf look-and-feel engine to eliminate legacy desktop styling, resulting in a sleek, dark, high-contrast user interface.
